# Comandos Básicos do Vim

### Movimentação:
- `h`: vai para **esquerda**
- `j`: vai para **baixo**
- `k`: vai para **cima**
- `l`: vai para **direita**

### Exclusão:
- `d`: delete
- `dd`: deleta a **linha**
- `dw`: deleta uma **palavra**
- `diw`: deleta a **palavra inteira**
- `D`: deleta antes do cursor
- `u`: desfaz (undo)
- `Ctrl + r`: refaz (redo)
- `c`: semelhante ao `d`, mas entra no modo de inserção
- `C`: deleta o restante da linha a partir do cursor e entra no modo de inserção

### Copiar e Colar:
- `y`: copia (yank)
- `p`: cola (paste)

### Outros Comandos:
- `cw`: apaga a palavra e entra no modo de inserção
- `0`: vai para o início da linha
- `$`: vai para o final da linha
- `{}`: movimentação mais rápida entre blocos de código (como `j` e `k` acelerados)
- `v`: entra no modo visual
